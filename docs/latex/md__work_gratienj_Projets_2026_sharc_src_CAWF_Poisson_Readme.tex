

Here, we utilize Arcane\+FEM to solve the Poisson equation, which is a fundamental elliptic partial differential equation (PDE). The provided code demonstrates a straightforward implementation of a 2D/3D unstructured mesh Galerkin finite element method (FEM) solver on arbitary meshes. Although we shall explain here only 2D for keeping the text simple.

The Poisson equation is encountered in various physical scenarios, including heat conduction, substance diffusion, membrane elasticity, inviscid fluid flow, electrostatics, twisting of elastic rods, and water waves. It serves as a vital tool for modeling and understanding these phenomena.\hypertarget{md__work_gratienj_Projets_2026_sharc_src_CAWF_Poisson_Readme_autotoc_md50}{}\doxysection{Theory}\label{md__work_gratienj_Projets_2026_sharc_src_CAWF_Poisson_Readme_autotoc_md50}

The 2D Poisson\textquotesingle{}s equation is solved for a closed meshed domain \$\textbackslash{}\+Omega$^\wedge$h\$ in order to know the solution \$u(x,y)\$ within the domain. The equation reads

\$\$\textbackslash{}frac\{\textbackslash{}partial\}\{\textbackslash{}partial x\}\textbackslash{}left( \textbackslash{}frac\{\textbackslash{}partial u\}\{\textbackslash{}partial x\} \textbackslash{}right) + \textbackslash{}frac\{\textbackslash{}partial\}\{\textbackslash{}partial y\}\textbackslash{}left( \textbackslash{}frac\{\textbackslash{}partial u\}\{\textbackslash{}partial y\} \textbackslash{}right) = \{\textbackslash{}mathcal\{f\}\} \textbackslash{}quad \textbackslash{}forall (x,y)\textbackslash{}in\textbackslash{}\+Omega$^\wedge$h \$\$

or in a more compact form

\$\$\textbackslash{}nabla$^\wedge$2 u = \{\textbackslash{}mathcal\{f\}\} \textbackslash{}quad \textbackslash{}forall (x,y)\textbackslash{}in\textbackslash{}\+Omega$^\wedge$h.\$\$

To complete the problem description, first type (Dirichlet) boundary conditions is applied to this problem\+:

\$u = 0.\+5 \textbackslash{}quad \textbackslash{}forall(x,y)\textbackslash{}in\textbackslash{}partial\textbackslash{}\+Omega$^\wedge$h\+\_\+\{\textbackslash{}text\{horizontal\}\}\textbackslash{}subset\textbackslash{}partial \textbackslash{}\+Omega$^\wedge$h,\$

Finally the right hand side source is present within the domain

\$\{\textbackslash{}mathcal\{f\}\}=5.\+5\$

\mbox{\hyperlink{structIn}{In}} this case the FEM variational formulation in \$H$^\wedge$1\+\_\+\{0\}(\textbackslash{}\+Omega) \textbackslash{}subset H$^\wedge$1\{\textbackslash{}\+Omega\}\$ reads

search FEM trial function \$u$^\wedge$h(x,y)\$ satisfying

\$\$-\/ \textbackslash{}int\+\_\+\{\textbackslash{}\+Omega$^\wedge$h\}\textbackslash{}nabla u$^\wedge$h \textbackslash{}nabla v$^\wedge$h + \textbackslash{}int\+\_\+\{\textbackslash{}partial\textbackslash{}\+Omega\+\_\+N\} (\textbackslash{}overline\{q\} \textbackslash{}cdot \textbackslash{}mathbf\{n\}) v$^\wedge$h + \textbackslash{}int\+\_\+\{\textbackslash{}\+Omega$^\wedge$h\}\{\textbackslash{}mathcal\{f\}\} v$^\wedge$h = 0 \textbackslash{}quad \textbackslash{}forall v$^\wedge$h\textbackslash{}in H$^\wedge$1\+\_\+0(\textbackslash{}\+Omega$^\wedge$h)\$\$

given

\$u$^\wedge$h=0.\+0 \textbackslash{}quad \textbackslash{}forall (x,y)\textbackslash{}in\textbackslash{}partial\textbackslash{}\+Omega$^\wedge$h\+\_\+\{\textbackslash{}text\{boundary\}\}\$,

\$\textbackslash{}int\+\_\+\{\textbackslash{}\+Omega$^\wedge$h\+\_\+\{\textbackslash{}text\{N\}\}\}(\textbackslash{}mathbf\{q\} \textbackslash{}cdot \textbackslash{}mathbf\{n\}) v$^\wedge$h=0\$ since no Neumann BC is present,

\$\textbackslash{}int\+\_\+\{\textbackslash{}\+Omega$^\wedge$h\}\{\textbackslash{}mathcal\{f\}\} v$^\wedge$h=5.\+5\$, and\hypertarget{md__work_gratienj_Projets_2026_sharc_src_CAWF_Poisson_Readme_autotoc_md52}{}\doxysection{The code}\label{md__work_gratienj_Projets_2026_sharc_src_CAWF_Poisson_Readme_autotoc_md52}
