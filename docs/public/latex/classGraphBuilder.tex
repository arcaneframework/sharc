\doxysubsection{Detailed Description}
\mbox{\hyperlink{namespaceArcane}{Arcane}} graph builder. 

\begin{DoxyAuthor}{Author}
Daniele A. Di Pietro \href{mailto:daniele-antonio.di-pietro@ifp.fr}{\texttt{ daniele-\/antonio.\+di-\/pietro@ifp.\+fr}}
\end{DoxyAuthor}
\textbackslash{}\+Graph\+Builder is a helper class intended to facilitate the creation of the \mbox{\hyperlink{namespaceArcane}{Arcane}} graph. The creation of the graph requires three steps\+: the definition of the link descriptors which characterize the links. A \textbackslash{}\+Link\+Descriptor is defined by the number of items connected by the link together with their type; the addition of the links. A link is created by passing to \textbackslash{}add\+Link the corresponding \textbackslash{}\+Link\+Data; the construction of the graph, which is carried out by the member \textbackslash{}build.

Build the graph. 

The graph description is provided in \mbox{\hyperlink{namespaceArcane}{Arcane}} as an array of \textbackslash{}\+Int64. The aim of this member is to build it from the information provided by the user. The array is composed of two parts\+: a headline, where the types of links appearing in the graph are described. A link description is composed of the number of items connected by the link together with their types. The following format is used\+: \hypertarget{classGraphBuilder_autotoc_md31}{}\doxysubsection{of descriptors}\label{classGraphBuilder_autotoc_md31}
$\vert$ \# of items in descriptor 1 $\vert$ type of item\+\_\+1 $\vert$ type of item\+\_\+2 $\vert$ ... $\vert$ type of item\+\_\+n1 $\vert$ \# of items in descriptor 2 $\vert$ type of item\+\_\+1 $\vert$ type of item\+\_\+2 $\vert$ ... $\vert$ type of item\+\_\+n2 ... a body, containing the groups of links. The following format is used \hypertarget{classGraphBuilder_autotoc_md32}{}\doxysubsection{of groups}\label{classGraphBuilder_autotoc_md32}
$\vert$ \# of links in group 1 $\vert$ descriptor\textquotesingle{}s progressive number $\vert$ link\+\_\+1 $\vert$ link\+\_\+2 $\vert$ ... $\vert$ link\+\_\+n1 $\vert$ \# of links in group 2 $\vert$ descriptor\textquotesingle{}s progreesive number $\vert$ link\+\_\+1 $\vert$ link\+\_\+2 $\vert$ ... $\vert$ link\+\_\+n2 ... A link is simply described by the (list of the) unique ids of the connected items.
