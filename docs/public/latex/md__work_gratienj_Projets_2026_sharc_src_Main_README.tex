Le module {\ttfamily Exa\+Di\+Bench} fournit une {\bfseries{batterie de benchmarks géométriques}} pour tester et valider les fonctionnalités de maillage et de géométrie d\textquotesingle{}Arc\+Geo\+Sim/\+Arcane. Il s\textquotesingle{}agit principalement de \+:


\begin{DoxyItemize}
\item déformer un maillage (déplacement aléatoire contrôlé des nœuds intérieurs),
\item recalculer les grandeurs géométriques (volumes, surfaces, centres),
\item vérifier la cohérence des groupes de bord ({\ttfamily Outer\+Faces}, nœuds de bord),
\item évaluer l\textquotesingle{}impact des déformations sur les mesures globales (volume total, surface totale, centre global).
\end{DoxyItemize}

Les cas de test correspondants se trouvent dans {\ttfamily test/\+Exa\+Di\+Bench/} (cube 3D, géométrie SPE11c, anneaux, etc.).\hypertarget{md__work_gratienj_Projets_2026_sharc_src_Main_README_autotoc_md109}{}\doxysubsection{Fonctionnalités}\label{md__work_gratienj_Projets_2026_sharc_src_Main_README_autotoc_md109}
