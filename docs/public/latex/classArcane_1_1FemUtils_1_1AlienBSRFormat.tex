\doxysubsection{Detailed Description}
A class for assembling Block Sparse Row (BSR) matrices from mesh data. 

The {\ttfamily \mbox{\hyperlink{classArcane_1_1FemUtils_1_1BSRFormat}{BSRFormat}}} class is designed to construct and manage BSR matrices using mesh connectivity and degree-\/of-\/freedom (DoF) information.

\mbox{\hyperlink{structIn}{In}} 2D, the sparsity is computed based on node-\/face connectivity, while in 3D, node-\/node connectivity is used.

\begin{DoxyNote}{Note}
This class uses \mbox{\hyperlink{namespaceArcane}{Arcane}}\textquotesingle{}s accelerator api and will use GPU is possible. It uses a {\ttfamily \mbox{\hyperlink{classArcane_1_1FemUtils_1_1AlienBSRMatrix}{Alien\+BSRMatrix}}} under the hood for representation and operations.

This class keep a reference on the instance of Fem\+Do\+FOn\+Nodes used in the constructor. So this instance of Fem\+Do\+FOn\+Nodes must live until the instance of this class is destroyed. 
\end{DoxyNote}
