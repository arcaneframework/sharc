Definitions des operateurs. 

\begin{DoxyDate}{Date}

\end{DoxyDate}
\begin{DoxyAuthor}{Author}
Xavier TUNC
\end{DoxyAuthor}
Definition de cinq classes pour traiter toutes les operations courantes sur des tableaux. ~\newline
La classe \mbox{\hyperlink{classBinaryAdd}{Binary\+Add}} est la classe utilisee dans le cas d\textquotesingle{}une addition (membre a membre) entre deux tableaux/expressions. ~\newline
La classe \mbox{\hyperlink{classBinaryMinus}{Binary\+Minus}} est la classe utilisee dans le cas d\textquotesingle{}une soustraction (membre a membre) entre deux tableaux/expressions. ~\newline
La classe \mbox{\hyperlink{classBinaryMultiply}{Binary\+Multiply}} est la classe utilisee dans le cas d\textquotesingle{}une multiplication (membre a membre) entre deux tableaux/expressions. ~\newline
La classe \mbox{\hyperlink{classBinaryDiv}{Binary\+Div}} est la classe utilisee dans le cas d\textquotesingle{}une division (membre a membre) entre deux tableaux/expressions. ~\newline
La classe \mbox{\hyperlink{classBinaryPow}{Binary\+Pow}} est la classe utilisee dans le cas d\textquotesingle{}un tableau eleve a la puissance n. ~\newline
Si on effectue une operation entre deux tableaux/expressions, cette operation s\textquotesingle{}effectuera toujours membre a membre. ~\newline
Si on effectue une operation entre un tableau/expression et un scalaire, cette operation s\textquotesingle{}effectuera sur tous les membres du tableau/expression. ~\newline
