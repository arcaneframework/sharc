17 // Outil pour obtenir une delegation aupres du ArcGeoSim::ModuleMaster
18 // Les fonctionnalites deleguees sont :
19 // - Gestion de l'affichage du pas de temps (IIterationInformation)
20 // - Gestion de l'avancement du temps (INextTimeComputer)
21 // - Gestion de points d'entrees extra (IExtraEntryPoints)
22 //
23 // Par exemple :
24 //
25 // class MyTimeInformation
26 // : public ArcGeoSim::Delegate<ArcGeoSim::IIterationInformation>
27 // {
28 // public:
29 // void timeStepInformation(Arcane::Integer nb_loop) { ... }
30 // };
31 //
32 // MyTimeInformation sera le delegue pour l'affichage des informations dans
33 // la time-loop
34 //
35 // Attention, on peut avoir plusieurs delegues mais seul le premier enregistre
36 // (i.e. instance cree) sera le delegue effectif. Pour changer de delegation,
37 // on appelera la methode claimDelegation().
